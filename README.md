## Getting started

```bash
$ poetry install

# Test Installation
$ poetry run python3 -c "import sqlalchemy_models; print(sqlalchemy_models)"
```

## Running tests

```bash
$ poetry run test
```

Usage:

```bash
$ pre-commit install

$ pre-commit run -a
```
